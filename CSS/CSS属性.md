# CSS属性

## margin



### margin-bottom

**定义和用法**

margin-bottom 属性设置元素的下外边距。

**注释：**允许使用负值。



**CSS 语法**

```
margin-bottom: length|auto|initial|inherit;
```

**属性值**

| 值       | 描述                                                         |
| :------- | :----------------------------------------------------------- |
| auto     | 浏览器计算下外边距。                                         |
| *length* | 规定以具体单位计的下外边距值，比如像素、厘米等。默认值是 0px。 |
| *%*      | 以包含元素宽度的百分比指定下外边距。                         |
| inherit  | 规定应该从父元素继承下外边距。                               |
